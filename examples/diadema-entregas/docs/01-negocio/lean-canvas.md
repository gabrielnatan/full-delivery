# Lean Canvas

## Problema
Taxas abusivas dos apps nacionais; restaurantes perdem margem.

## Segmentos de clientes
Restaurantes MEI; entregadores autônomos; moradores de Diadema.

## Proposta de valor única
O delivery que conhece Diadema.

## Solução
App cliente + painel restaurante + app entregador + API com Pix.

## Canais
Prospecção presencial; WhatsApp; Instagram local.

## Fontes de receita
Comissão 12% + R$ 1 fixo por pedido.

## Estrutura de custos
Cloud, dev, marketing local, suporte.

## Métricas-chave
GMV, restaurantes ativos, NPS, tempo de entrega.

## Vantagem injusta
Rede local de restaurantes fundadores.
