# Decisões de Arquitetura (ADR)

> Architecture Decision Records — uma decisão por arquivo, no formato `ADR-XXX-slug.md`.
> Não edite decisões antigas: crie uma nova que a substitua (`supersedes`).

## Template

```markdown
---
id: ADR-001
type: decision
status: accepted        # proposed | accepted | deprecated | superseded
supersedes: null          # ID de outro ADR, se aplicável
created: 2026-07-09
updated: 2026-07-09
tags: []
---

# Título da decisão

## Contexto
(qual problema/força motivou a decisão)

## Decisão
(o que foi decidido)

## Alternativas consideradas
(o que mais foi cogitado e por que foi descartado)

## Consequências
(trade-offs — o que ganhamos e o que abrimos mão)
```
