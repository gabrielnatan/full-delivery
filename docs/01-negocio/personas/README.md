# Personas

Um arquivo por persona, no formato `PERSONA-XXX-slug.md`. Cada arquivo vira um card na
tela **Personas** do gerenciador.

## Template

```markdown
---
id: PERSONA-001
type: persona
status: active
created: 2026-07-09
updated: 2026-07-09
tags: []
---

# Nome da Persona

## Perfil
(idade, tipo de negócio/uso, contexto, onde está)

## Dores
- dor 1
- dor 2

## Comportamento atual
(como resolve hoje, que ferramentas/apps usa)

## O que faria migrar pra gente
(gatilho de troca)
```
